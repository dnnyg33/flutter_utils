# flutter_utils

A collection of shared Flutter widgets and utilities — networking primitives (`AsyncData`, `AsyncOperation`, `PaginatedAsyncData`), styling helpers, and reusable widgets.

## Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  flutter_utils:
    git:
      url: https://github.com/dnnyg33/flutter_utils.git
```

To pin to a specific commit or tag:

```yaml
dependencies:
  flutter_utils:
    git:
      url: https://github.com/dnnyg33/flutter_utils.git
      ref: v0.0.1
```

Then run:

```sh
flutter pub get
```

## Usage

```dart
import 'package:flutter_utils/flutter_utils.dart';
```
